# MonoDex

A web application for searching Pokemon cards by Pokemon name. Users can provide a Pokemon name and retrieve a list of all Pokemon cards featuring that Pokemon by querying a database and Pokemon API.

## Features

- Search Pokemon cards by name
- Display all cards featuring a selected Pokemon
- Integration with Pokemon database and public APIs

## Getting Started

### Prerequisites
- Node.js 18+ 
- npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Build

```bash
npm run build
npm start
```
